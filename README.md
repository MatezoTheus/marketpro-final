# MarketPro Final

Sistema web de gestão para supermercados desenvolvido para projeto acadêmico.

## Funcionalidades

- Dashboard de vendas
- Gestão de produtos
- Controle de estoque
- Cadastro de clientes
- PDV (Ponto de Venda)
- Sistema de fidelidade e cashback
- Analytics inteligentes
- Histórico de vendas
- Exportação CSV
- Dark Mode

## Tecnologias

- HTML5
- TailwindCSS
- JavaScript
- Chart.js

## Como executar

Basta abrir o arquivo `index.html` no navegador.
